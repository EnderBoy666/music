# 这是一个点歌台
---
### 是的，这是一个十分正经的点歌台
*接下来是使用教程。*
* 首先，你需要打开终端，输入以下命令
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
* 现在，你已经创建好了运行环境。这个项目有两个文件需要启动，分别是`main.py`（用于点歌）和`play.py`用于放歌
所以，请运行
```bash
python main.py
```
```bash
python play.py
```
* 好的，现在你已经跑起来了，点歌位于8123端口，放歌位于8124端口
